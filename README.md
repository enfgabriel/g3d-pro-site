# G3D Pro

O G3D Pro é uma ferramenta para organizar serviços de impressão 3D, desde o cadastro do cliente até orçamento, pedido, estoque e produção.

## Acessando

1. Abra o site do G3D Pro.
2. Entre com seu e-mail e senha.
3. Se ainda não tiver conta, use a opção de cadastro.
4. Depois do login, você verá o painel principal com os atalhos do sistema.

## Minha loja

Use **Minha loja** para cadastrar as informações do seu negócio:

- nome da loja;
- responsável;
- CNPJ ou CPF;
- WhatsApp;
- e-mail;
- site ou Instagram;
- endereço;
- logo;
- observação padrão dos orçamentos.

Essas informações aparecem nos documentos gerados pelo sistema.

## Parâmetros

Use **Parâmetros** para configurar a base dos cálculos de orçamento:

- custo de energia;
- consumo médio da impressora;
- custo por hora de máquina;
- custo padrão por grama;
- margem;
- taxa mínima;
- pós-processamento;
- embalagem padrão.

Quanto melhor esses dados estiverem preenchidos, mais preciso será o orçamento.

## Estoque

Use **Estoque** para cadastrar filamentos, resinas e insumos.

Para cada item, informe material, cor, marca, quantidade disponível, quantidade mínima e custo por g/ml. Quando um orçamento usa um item do estoque, o sistema pode calcular o valor com base nesse custo.

## Clientes

Use **Clientes** para manter os dados comerciais dos seus clientes organizados:

- nome;
- empresa;
- CPF/CNPJ;
- e-mail;
- telefone;
- WhatsApp;
- cidade e estado;
- endereço;
- observações.

## Orçamentos

Use **Orçamentos** para criar propostas para seus clientes.

1. Escolha o cliente.
2. Informe o projeto.
3. Adicione os itens do orçamento.
4. Selecione o material do estoque, quando existir.
5. Informe peso, tempo de impressão, pós-processamento, embalagem e margem.
6. Confira o valor calculado.
7. Gere o documento e salve como PDF.
8. Quando o cliente aprovar, transforme o orçamento em pedido.

## Pedidos

Use **Pedidos** para acompanhar o que já foi aprovado.

Você pode controlar status, prioridade, valor, data de entrega e observações do serviço.

## Produção

Use **Produção** para acompanhar a fila de impressão.

Cada produção pode ter pedido vinculado, impressora, material, cor, peso consumido, tempo e status. Ao finalizar uma produção, o sistema ajuda a manter o estoque atualizado.

## Central de notícias

A **Central de notícias** aparece no dashboard e mostra comunicados, atualizações, dicas e parceiros.

Somente o administrador do G3D Pro pode publicar ou editar conteúdos nessa área. Usuários comuns apenas visualizam as publicações ativas.

## Segurança de uso

Cada conta acessa o seu próprio ambiente. Clientes, orçamentos, pedidos, estoque, parâmetros e produção ficam separados por usuário.
